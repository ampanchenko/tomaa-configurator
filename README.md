# TOMAA — онлайн-конфигуратор браслета

Интерактивный конструктор браслетов с шармами для бренда **tomaa** (tomaa.ru).
Посетитель выбирает основу (тип, материал, цвет), надевает до 5 шармов с миксом
золота и серебра, и оформляет заказ — состав уходит в корзину Tilda.

**Демо:** https://ampanchenko.github.io/tomaa-configurator/

## Возможности
- Основа: узкий 10 мм / широкий 15 мм, один оборот / на два оборота, кожа/бархат, 5–6 цветов.
- До 5 шармов в ряд на каждый ремешок, микс золота и серебра, перетаскивание мышью.
- Полный каталог шармов (74 шт.) с фильтром по категориям.
- Реальные названия, цены и Tilda UID — кнопка «Оформить заказ» кладёт товары в корзину Tilda.
- Дизайн в стиле tomaa.ru (Montserrat, чёрно-белая палитра).

## Встраивание в Tilda
1. Адрес демо выше — это готовая страница.
2. На странице Tilda добавьте блок «HTML» (T123) и вставьте код из [`tilda-embed.html`](tilda-embed.html),
   заменив адрес на свой (по умолчанию — GitHub Pages этого репозитория).

Подробности — в [`ИНСТРУКЦИЯ.md`](ИНСТРУКЦИЯ.md).

## Структура
- `index.html` — сам конфигуратор (самодостаточный, HTML+CSS+JS).
- `assets/` — оптимизированные картинки (браслеты в цветах, шармы, превью).
- `tilda-embed.html` — сниппет для вставки на Tilda через iframe.
- `build_assets.py`, `recolor_bases.py`, `fetch_charm_previews.py` — сборка картинок из
  исходников (папка `images/` в репозиторий не входит).
